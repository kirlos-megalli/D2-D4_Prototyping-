# Autonomous Driving Prototype

## Overview
This project focuses on developing an autonomous vehicle prototype capable of driving autonomously on a predefined track using line detection and obstacle avoidance techniques.

The project is part of the Systems Engineering & Prototyping course and combines SysML modeling, simulation, and embedded systems development.

---

## Project Goals
The autonomous vehicle should be able to:

- Follow a lane/line autonomously
- Detect and avoid obstacles
- Optimize driving speed
- Handle different driving routes
- Drive in oval and figure-eight patterns
- Perform autonomous parking
- Simulate behavior using Tinkercad
- Apply Systems Engineering methods and SysML diagrams

---

## Features
- Line following
- Obstacle detection
- Autonomous routing
- Speed optimization
- Parking functionality
- Figure-eight driving mode
- Tinkercad simulation
- SysML/UML documentation
